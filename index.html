Guía Rápida para No-Root (ADB Setup)

  1. Habilita Opciones de Desarrollador: Ajustes > Acerca del teléfono > Toca "Número de compilación" 7 veces.
  2. Activa Depuración USB: Opciones de Desarrollador > Depuración USB.
  3. Instala ADB en PC: Descarga desde Android SDK.
  4. Ejecuta comandos: adb shell [comando] o usa el script con adb shell -x sh script.sh.
  5. Para Termux: Usa pkg install android-tools para ADB local (requiere wireless debugging).
