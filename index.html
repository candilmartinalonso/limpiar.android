APEX Optimizer v4.0

Generador de Script Termux de Optimización Extrema

1. Propósito del Script: Optimización Extrema (v4.0)

Esta versión **APEX Optimizer v4.0** está diseñada para una limpieza y optimización del rendimiento de Android no root, llevando al límite las capacidades de Termux. A diferencia de las versiones anteriores, esta utiliza comandos find altamente optimizados (con ejecución paralela -exec {} +) y consolida la limpieza de caché, logs, archivos temporales, `apk`, `deb`, e incluso basura de archivos multimedia como `.nomedia` y miniaturas.

El script incluye una purga agresiva de las cachés de Termux (`pkg clean`, `pip cache purge`, `npm cache clean`), una limpieza de procesos 'zombie' y una triple ejecución de `sync` para liberar la máxima cantidad de memoria RAM posible de forma segura. Todas las acciones se registran con precisión para garantizar la estabilidad y la transparencia.

3. Seguridad, Uso y Mejoras (v4.0)

Seguridad y Rutinas No Destructivas

  • **Operación No Root:** El script solo usa comandos estándar de Termux y Android, garantizando que no puede dañar archivos del sistema operativo principal.
  • **Limpieza Ultra-Targeted:** La limpieza de archivos está restringida a tipos de archivos temporales conocidos y a la carpeta de almacenamiento compartido (`$STORAGE_PATH`).
  • **Redirección de Logs:** Se utiliza `exec > >(tee "$LOG_FILE") 2>&1` para redirigir toda la salida a la consola **y** a un archivo de registro (log) datado para máxima transparencia.
  • **Recuperación de RAM:** La triple ejecución de `sync` es el método no root más agresivo para liberar la caché de memoria del kernel de Android.

Uso y Características Adicionales

  1. **Habilitar Almacenamiento:** Ejecuta `termux-setup-storage` una vez para dar acceso al almacenamiento compartido (`/sdcard`).
  2. **Guardar Script:** Copia el código de arriba y guárdalo: `nano optimizer.sh`. Pega el código y guarda (Ctrl+X, Y, Enter).
  3. **Permisos:** Dale permisos de ejecución: `chmod +x optimizer.sh`.
  4. **Ejecutar:** Lanza el script: `./optimizer.sh`.
  5. **Automatización:** Puedes usar cron jobs (con `pkg install cronie`) para ejecutarlo automáticamente por la noche: `0 3 * * * /data/data/com.termux/files/home/optimizer.sh` (Ejecuta a las 3 AM diariamente).
