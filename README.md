## 📋 Descripción

**Limpiar Android** es una herramienta web interactiva diseñada para ayudar a los usuarios a optimizar y limpiar sus dispositivos Android de manera eficiente. La aplicación web proporciona una interfaz intuitiva con recomendaciones personalizadas, consejos de optimización y herramientas para mejorar el rendimiento de dispositivos Android.

### ✨ Características Principales

- 🔧 **Análisis de Rendimiento**: Evaluación completa del estado del dispositivo
- 🗑️ **Limpieza Inteligente**: Identificación de archivos innecesarios y cache
- ⚡ **Optimización Automática**: Sugerencias personalizadas de mejora
- 📱 **Interfaz Responsiva**: Compatible con todos los dispositivos
- 🎨 **Diseño Moderno**: UI/UX optimizada para la mejor experiencia de usuario
- 🌐 **Multiplataforma**: Accesible desde cualquier navegador web

## 🚀 Instalación y Uso

### Acceso Directo

La aplicación está disponible online en: [https://candilmartinalonso.github.io/limpiar.android/](https://candilmartinalonso.github.io/limpiar.android/)

### Instalación Local

1. **Clona el repositorio**:
   ```bash
   git clone https://github.com/candilmartinalonso/limpiar.android.git
   cd limpiar.android
   ```

2. **Abre en tu navegador**:
   - Abre el archivo `index.html` directamente en tu navegador preferido
   - O utiliza un servidor local (recomendado):
     ```bash
     # Con Python 3
     python -m http.server 8000
     
     # Con Node.js (live-server)
     npx live-server
     
     # Con PHP
     php -S localhost:8000
     ```

3. **Accede a la aplicación**:
   - Navega a `http://localhost:8000` en tu navegador
